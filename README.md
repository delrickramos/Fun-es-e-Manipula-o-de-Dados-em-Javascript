# Funções e Manipulação de Dados em JavaScript

Este projeto é uma aplicação web simples desenvolvida com **HTML**, **CSS** e **JavaScript**, que demonstra o uso de diferentes tipos de **funções** e **manipulação de dados** em JavaScript.

---

## Objetivo

O principal objetivo deste projeto é **praticar e demonstrar conceitos de funções em JavaScript**, incluindo:

- Funções **nomeadas**, **anônimas** e **arrow functions**  
- Uso de **callbacks** (funções passadas como parâmetro)  
- **Manipulação de arrays**, como filtragem e cálculo de média  

---

## Funcionalidades

A aplicação contém três seções principais:

### 1. Operações básicas  
Permite inserir dois números e escolher uma operação matemática para calcular o resultado:
- Soma  
- Subtração  
- Multiplicação  
- Divisão  

### 2. Manipulação de arrays  
Permite inserir uma lista de números e:
- Filtrar apenas os **números pares**  
- Calcular a **média** dos valores informados  

### 3. Callback  
Demonstra o uso de **funções de callback** aplicadas a operações matemáticas.

---

## Tecnologias utilizadas

- **HTML5** → Estrutura do projeto  
- **CSS3** → Estilização e layout responsivo  
- **JavaScript (ES6+)** → Lógica e manipulação de dados
